# 合同测试样例说明

用于手机端「本地文件导入」真实链路测试。请使用 **`.txt`**（当前版本只有文本文件会把正文传给分析服务；PDF/图片只会传元数据，容易落到模板兜底）。

## 如何拷到手机

1. 把本目录下需要的 `.txt` 传到手机（隔空投送 / 微信文件传输 / USB / 网盘均可）。
2. 建议放到「文件」App 里易找的位置，例如「下载」或自建「合同样例」文件夹。
3. 打开 App → 导入合同 → 选择本地文件 → 选中对应 `.txt`。
4. 确保分析服务已启动，且设置页里的服务地址/API Key 可用。

## 样例一览

| 文件 | 场景 | 预期风险倾向 | 重点可观察条款 |
|------|------|--------------|----------------|
| `01_租房合同_高风险.txt` | 住房租赁 | 高 | 自动续约 2 年、押金任意扣、单方解释权、管辖偏甲方 |

| `04_培训协议_高风险.txt` | 职业培训 | 高 | 开课后不退费、就业承诺免责、禁差评 |
| `05_兼职劳务_中风险.txt` | 兼职劳务 | 中 | 节假日不加薪、扣薪离职、意外险非强制 |
| `06_服务外包_低风险.txt` | 设计外包 | 低 | 分期付款、IP 结清后转移、对等解约 |
| `07_会员订阅_高风险.txt` | 自动续费会员 | 高 | 默认续费、可改价、几乎不退、数据归平台 |
| `08_家政服务_中风险.txt` | 家政保洁 | 中 | 过期作废、赔偿上限过低、管辖偏乙方 |
| `09_购机合约_高风险.txt` | 通讯购机捆绑 | 高 | 36 个月锁网、口头规则优先、个人信息概括授权 |
| `10_实习协议_低风险.txt` | 大学生实习 | 低 | 工时合理、可写简历、解约对等 |

另：仓库 `doc/test_contract_sample.txt` 是办公场地租赁综合样例（偏中高风险：逾期违约金、押金退还表述偏模糊）。

## 测试建议

- **对比组**：先测 `01` 与 `02`、或 `03` 与 `10`，看高低风险差异是否被识别。
- **场景覆盖**：租房 / 实习 / 培训 / 订阅 / 服务，各测至少 1 份。
- **答辩演示**：若弱网，可改用 App 内置「演示样例」；本目录样例更适合验证真实 LLM 分析。

> 样例为虚构测试文本，不含真实个人敏感信息完整号段，仅供功能验证，不构成法律意见。
